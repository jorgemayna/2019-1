El programa se encunetra en el archivo shell.c.
Estan implentados los comandos cd,ls,mkdir,pwd,rm


En el bash creado les puse otros nombres a los comandos:
cd -- cd
ls -- dir
mkdir --crtdir     (create dir)
pwd -- ntd         (name this dir)
rm -- dl           (delete)


el comando dl acepta 2 posibles argumentos:

-d  :  para borrar directorios
-r  :  para borrar recursivamente
